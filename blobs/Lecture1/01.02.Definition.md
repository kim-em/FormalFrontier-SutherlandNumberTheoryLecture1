**Definition 1.2.** An *absolute value* on a field $k$ is a map $\lvert \cdot \rvert \colon k \to \mathbb{R}_{\geq 0}$ such that for all $x, y \in k$ the following hold:

1. $\lvert x \rvert = 0$ if and only if $x = 0$;
2. $\lvert xy \rvert = \lvert x \rvert \lvert y \rvert$;
3. $\lvert x + y \rvert \leq \lvert x \rvert + \lvert y \rvert$.

If the stronger condition

4. $\lvert x + y \rvert \leq \max(\lvert x \rvert, \lvert y \rvert)$

also holds, then the absolute value is *nonarchimedean*; otherwise it is *archimedean*.
